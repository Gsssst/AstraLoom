## Why

The product name `Auto-Research-DS` is descriptive but not memorable as a user-facing brand. The project is now a broader research workspace spanning papers, ideas, experiments, AI discussion, and writing, so it needs a more distinctive identity.

## What Changes

- Rename the current user-facing product identity to `AstraLoom`.
- Update frontend brand surfaces, backend app metadata, LaTeX generated author metadata, and current documentation references.
- Redesign the home page around an AstraLoom visual concept: research constellations, woven evidence lines, and direct workflow entry points.
- Keep route paths, APIs, database schema, and authentication behavior unchanged.

## Capabilities

### New Capabilities

- `product-branding`: User-facing product identity, homepage brand presentation, and current documentation naming.

### Modified Capabilities

None.

## Impact

- Frontend: home page, login/register brand text, command palette copy, and tests.
- Backend: app name, generated LaTeX author string, arXiv User-Agent.
- Documentation: current README, introduction, user manual, and OpenSpec project overview.
