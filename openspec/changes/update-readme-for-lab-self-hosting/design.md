## Context

AstraLoom is intended as a self-hosted lab workspace. Each research group should deploy its own instance, keep papers/API keys/private notes under its own control, and use the system for internal literature management and AI-assisted research workflows.

## Approach

- Lead with the self-hosted lab use case in both Chinese and English.
- Avoid language that implies a public hosted website or centralized multi-tenant SaaS.
- Keep setup commands short and actionable, with bilingual section headings where useful.
- Include a GitHub publishing checklist so sensitive files are not uploaded.

## Non-Goals

- Do not add deployment automation.
- Do not change `.gitignore` unless verification finds an unsafe gap.
- Do not add screenshots or marketing copy in this change.
