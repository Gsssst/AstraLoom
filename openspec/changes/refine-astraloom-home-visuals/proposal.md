## Why

The newer AstraLoom home page visual treatment made the hero and lower workflow area feel less aligned with the user's preferred earlier style. Restoring the previous home page composition while keeping the AstraLoom product name preserves the brand rename without introducing unwanted visual changes.

- Restore the earlier particle-based home page hero, quick actions, stats, feature cards, and footer layout.
- Keep `AstraLoom` as the user-facing product name on the restored home page.
- Update frontend contract coverage so tests match the restored home page visual hooks.
