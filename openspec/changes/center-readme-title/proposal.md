## Why

GitHub renders the top Markdown `# AstraLoom` heading left-aligned, while the rest of the README hero content is centered. Centering the title makes the repository landing page visually consistent.

## What Changes

- Replace the README's top Markdown heading with a GitHub-renderable centered HTML heading.
- Keep the existing README content, badges, links, and positioning unchanged.

## Capabilities

### New Capabilities

- None.

### Modified Capabilities

- `product-branding`: README GitHub presentation shall center the primary product title with the rest of the hero block.

## Impact

- Affected docs: `README.md`.
- Affected spec: `product-branding`.
- No runtime, API, dependency, or data changes.
